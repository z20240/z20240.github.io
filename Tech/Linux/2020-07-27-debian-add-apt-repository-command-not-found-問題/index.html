# [Debian] add-apt-repository command not found 問題

add-apt-repository 是用來更新 apt-get 套件來源的指令。

在比較新的 Ubuntu 或是 Debian 版本，這可能不是在初始預設便有的指令集中。

我們可以使用以下指令來安裝這個指令套件。

$ sudo apt-get install software-properties-common
