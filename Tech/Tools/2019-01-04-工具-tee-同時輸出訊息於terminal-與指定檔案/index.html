# [工具] [tee] 同時輸出訊息於 Terminal 與指定檔案

這篇作為一個紀錄，最近工作由於在追一個 bug ，希望將在小黑窗 (Terminal) 上的訊息能夠也寫一份在檔案中，因此就在網路上找到了這個方法 — 使用 tee

tee 的功用便是將數據導向到文件中，最簡單的使用方式如下

tee <out file>

當然一般時候我們都會搭配 shell 的 pipline 一併使用，例如

ls | tee out.txt   # 將 ls 的結果寫到 out.txt

所以這次對這個需求使用方式變如下

command | tee -a <路徑位置>

其中，這邊下 -a 的原因是我希望 以累加 (append) 的方式，將資料加入 file 當中 。
